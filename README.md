# Granite ResearchAI

AI-Powered Research Agent using IBM Granite, Docling and LangFlow.

## Domain
Education and Research

## Problem Statement
Research Agent

## Technologies Used
- IBM Granite Models
- IBM watsonx.ai
- LangFlow
- Docling
- RAG
- IBM Cloud Lite

## Features
- Research Paper Analysis
- Literature Summarization
- Research Data Retrieval
- Citation Support
- Research Gap Identification
- Automated Report Generation

## Architecture
Multi-Agent AI system consisting of:
- Document Processing Agent
- Knowledge Retrieval Agent
- Research Analysis Agent
- Report Generation Agent
